# 星際霸權 Star Hegemony

原創太空艦隊戰略遊戲 — 專制帝國 vs 民主聯邦。

GDD: [Notion](https://app.notion.com/p/3921f269eaea8108b6e7e7855ce4edcb)

## 技術棧

Canvas 2D + Vanilla JS(ES modules)+ Web Audio,零依賴 — 同 Tower Control 相同 pipeline。

## 執行

```bash
cd star-hegemony
python3 -m http.server 8080
# 開 http://localhost:8080
```

(ES modules 需要 http server,直接開 file:// 唔得)

## v0.2 — 戰役層 Campaign Update

### 戰役模式 🌌
- **星圖**:11 個星系、航線網絡、陣營控制;目標攻陷帝國首都奧丁,同時守住新特拉維夫
- **經濟**:星域收入、艦艇招募(船塢)、艦隊維修、戰場繳獲
- **補給線**:移動消耗補給,友方星域回補;斷補開戰士氣 -20
- **議會政治**:支持度影響每 5 回合撥款;跌穿 15 被解除指揮權(Game Over)
- **黑天鵝事件**:瘟疫、叛徒洩密、隕石帶、傭兵「幽靈」、邊境總督、奧圖退役、醜聞、補給船隊
- **戰後政治決策卡**:戰俘處置、戰場打撈、傳媒專訪、難民救援
- **帝國 AI**:雷因哈特喺星圖上圍攻聯邦星域;被擊敗會退守奧丁重建;攻打奧丁必定觸發同佢嘅決戰
- **停戰談判**:一次性凍結帝國 4 回合(支持度 -12)
- **自動存檔**:localStorage,主選單可繼續戰役

### 戰鬥新系統 ⚔️
- **軍官系統**(GDD 人物名冊):瑪雅(梯次防禦強化)、雷(斬首強化)、菲比(後勤折扣)、艾莎(策反)
- **策反**(7 鍵):每場一次,嘗試策反敵艦轉投;敵士氣低成功率更高
- **戰術直覺**:里昂被動——敵將發動戰術前 2 秒預警
- **老兵系統**:艦艇擊殺升級(最多 3 級,每級 +8% 傷害),金色軍階章
- **士氣崩潰兵變**:士氣 <20 艦艇可能擅自逃離戰場(雙方都會)
- **蕾雅・達克**:帝國士氣低迷時可能嘩變轉投(每戰役一次)
- **撤退**(8 鍵):有序脫離,戰役中保留殘存艦艇
- **戰史回放**:戰後 cinematic 回放成場會戰(3x 速度)
- **敵將人格**:雷因哈特(反制戰術/電光戰法/詐敗懲罰×2)、沃夫(進攻狂)、駐防司令(保守)

### 遭遇戰模式
單場 vs 雷因哈特,即 v0.1 玩法 + 全部新戰鬥系統。

## 檔案結構

- `js/main.js` — 遊戲迴圈、渲染、輸入、炮火/水雷/粒子、回放、策反/兵變/蕾雅
- `js/fleet.js` — 艦艇/艦隊/陣型/戰術/老兵/mods
- `js/tactics.js` — 六大戰術定義
- `js/ai.js` — 敵將人格 AI(雷因哈特/沃夫/駐防)
- `js/campaign.js` — 戰役層邏輯(星圖/經濟/回合/帝國AI/存檔,純邏輯無 DOM)
- `js/campaign-ui.js` — 星圖渲染、系統面板、modal 流程
- `js/events.js` — 黑天鵝事件 + 政治決策卡
- `js/roster.js` — 人物名冊(軍官加成)
- `js/tech.js` — 科技樹(船塢強化/後勤網絡/經濟改革,各 3 級)
- `js/ui.js` — 戰鬥 HUD
- `js/audio.js` — Web Audio 合成音效

## Debug

- `?auto=1&tactic=pincer&speed=8` — 自動遭遇戰(headless 測試)
- `?auto=1&mode=campaign` — 自動開戰役
- Node 模擬:battle sim + 全戰役 sim(見 scratchpad harness pattern),用嚟平衡測試
